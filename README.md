# LeCateringProject


[Demo](https://rkymtc.github.io/LeCateringProject/)

## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/rkymtc/LeCateringProject.git
```

Proje dizinine gidin

```bash
  cd LeCateringProject
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm start
```
